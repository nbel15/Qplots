Script commands:

  • Open terminal: Ctrl + T
  • Change permission: chmod +x path/to/script/scriptname.sh (eg. chmod +x Desktop/script-ITS.sh)
  • Run: ./script-ITS.sh
  • Modify script from command line: nano path/to/script/scriptname.sh (eg. nano Desktop/script-ITS.sh)
  • Save modifications: Ctrl + X --> yes --> Enter

Qiime2 commands:

  • Activate qiime2: source activate qiime2-2019.1 (Select the appropriate qiime environment from conda list (conda info --envs))
  • Import reference sequences: qiime tools import --input-path path/to/ref/seq99.fasta --output-path seq99.qza --type 'FeatureData[Sequence]'
  • Import taxonomy file: qiime tools import --input-path path/to/taxa/taxa99.txt --output-path taxa99.qza --type FeatureData[Taxonomy] --input-format HeaderlessTSVTaxonomyFormat

Prepare FASTQ files:

Paired-end amplicon sequencing provide two FASTQ files for each sample, usually these files are compressed in .gz and .zip format.

  • To uncompress .zip format: unzip path/to/file/filename.zip. This command create a new folder contains the compressed FASTQ files
  • To uncompress all the FASTQ file in a folder: gunzip path/to/new/folder/*.gz (*.gz: refer to all the files with .gz fomart)
  • Rename FASTQ folder: mv path/to/old/folder/old_name path/to/new/folder/fq (eg. mv Desktop/Glossinafastq Desktop/fq)
