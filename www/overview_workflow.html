Oplots provide a tipical workflow for analysis of microbiome data. Whitch contains the following steps:

  • Import Data: Determine the dataset to use as the starting point of the analysis.
  • Relative Abundance: Calculate relative abundance matrix and produce abundance bar chart for different taxonomy levels according to given categories.
  • Alpha Diversity: Calculate alpha diversity indices among samples and visualize indices in Boxplots with significance groups.
  • Beta Diversity: Calculate the beta-diversity between samples and visualize groupings of samples according to given categories.
  • Serial Group Comparison: Calculate non-parametric ANOVA (Kruskal-Wallis Rank Sum Test) and Fisher tests across non-paired input numerical variables over selected categorical variables.
  • Network Input: Generate OTU table with taxonomy for Co-occurence analysis in Cytoscape.

The analysis of the raw data could be done using a shell script (Check FASTQ Analysis section for more details). this script use Usearch11 to generate the OTU table, and Qiime2 (version 2019-1) to asign taxonomy based on SILVA/UNITE database and generate the midpoint tree.
