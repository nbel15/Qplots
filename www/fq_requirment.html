Installations

  • Operation system: Linux distributions (tested only in Ubuntu), Mac OS
  • USEARCH11: Install USEARCH version 11 32-bit (open access) or 64-bit (commercial licenses)
  • QIIME2: Install QIIME2 version 2019.1

Reference Database

  • Download and import in QIIME2 the appropriate reference sequences and taxonomy files from:
    • SILVA: For 16S rRNA amplicons
    • UNITE For ITS amplicons
  • Change Database path in Assign taxa section.

Remarque:

  • Uncompressed FASTQ file should be saved in one folder with fq as name

  • fq folder and the sh script should be stored in the same folder
